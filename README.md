# IT Portfolio

A modern, responsive portfolio website built with React and Tailwind CSS to showcase programming skills, projects, and contact information.

## Features

- **Responsive Design**: Fully responsive layout
- **Modern UI**: Clean and professional design with smooth animations and transitions
- **Skills**: Visual representation of programming languages and tools
- **Projects**: Featured projects with descriptions, technologies used, and live demo/GitHub links
- **Contact**: Personal contact
- **Optimized Performance**: Built with React and Tailwind CSS for fast loading times

## Tech Stack

- **React**: JavaScript library for building user interfaces
- **Tailwind CSS v4**: Utility-first CSS framework for styling
- **Vite**: Next-generation frontend tooling for fast development
- **Lucide React**: Beautiful icon library
- **JavaScript (JSX)**: Modern JavaScript with JSX syntax
